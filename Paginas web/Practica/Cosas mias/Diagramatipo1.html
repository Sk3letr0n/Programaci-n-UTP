Glosario 1
Acerca de
Para la estructura de la página web:
HTML: Indica el inicio y final de un documento HTML.
HEAD: Define la cabecera del documento.
BODY: Define el cuerpo del documento.
TITLE: Indica el título de la página.
TABLE: Agrega una tabla.
TR: Agrega una fila o un renglón en una tabla.
TD: Agrega una columna en una tabla.
Style: Agrega un estilo CSS dentro del documento.
Para formatear el texto
B: Para poner negrillas, ejemplo: Hola
I: Para poner cursiva, ejemplo: Que
U: Para poner subrayado, ejemplo: Tal
br: Indica un salto de linea, ejemplo: Siempre
que
P: Para escribir un párrafo
H1: Para un subtítulo de nivel 1.
H2: Para un subtítulo de nivel 2.
H3: Para un subtítulo de nivel 3.
SUP: Para poner un superíndice: 132
SUB: Para poner un subíndice: Ho
UL: Para hacer una lista no organizada.
OL: Para hacer una lista organizada.
LI: Son los items de la lista.
